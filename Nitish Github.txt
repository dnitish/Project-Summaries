1. https://github.com/dnitish/timesheetmanagement

This project is made with Angular [ Angular is a TypeScript-based free and open-source web application framework ] and Loopback [ LoopBack is a highly-extensible, open-source Node.js framework that enables you to create dynamic end-to-end REST APIs with little or no coding. ] and Angular Material for Design Components.

The entire project has been containerized in Docker with a MySQL database. Login Functionality implemented through NodeMailer.

The main focus of the project is to enable a timesheet management system for employees to list out tasks and ID them by labels, for eg. in progress, completed etc. There are 2 views, an admin view and the employee view. The admin can assign tasks to employees and has full view of their progress.
